1. 常见面试问题（可能涉及） 自我介绍 Tell me about your background and experience in data centers. 技术经验 What experience do you have with rack and stack? How do you handle fiber optic cables? What’s the difference between single-mode and multi-mode fiber? Can you explain the difference between SFP, SFP+, and QSFP? Describe your experience with ticketing systems like ServiceNow. Have you worked with Excel? Can you use pivot tables or formulas? 场景类问题 A server is not powering on after installation, how would you troubleshoot? You receive multiple urgent tickets at the same time, how do you prioritize? If a cable is mislabeled, what steps do you take to fix it? 软技能 / 团队 Are you comfortable working night shifts and weekends? Can you lift heavy equipment (servers, UPS, etc.)? Do you have leadership experience in a data center environment?

2. 技术知识要点 ✅ Rack & Stack 安装服务器、交换机、存储设备到机柜 注意气流方向、电源冗余、线缆管理 ✅ 光纤 & 网线 Single-mode fiber (SMF) → 长距离，黄色，使用激光 Multi-mode fiber (MMF) → 短距离，橙色/水蓝色，使用 LED SFP (Small Form-factor Pluggable) → 支持 1Gbps SFP+ → 支持 10Gbps QSFP → 支持 40Gbps/100Gbps ✅ Excel 技能 常用函数：VLOOKUP / IF / SUM / COUNTIF 数据透视表（Pivot Table） 整理机房资产清单 ✅ Ticketing System (ServiceNow) 工单创建、分配、关闭 记录设备更换、布线、服务器部署等 ✅ 故障排查思路 电源 → 网络 → 配置 → 软件 检查硬件灯、线缆、SFP 模块兼容性 📌

3. 英文回答示例 Q: Tell me about your data center experience. 👉 “I have over five years of experience working in data centers. My responsibilities included racking and stacking servers, running fiber and Ethernet cables, and managing equipment inventory. I also used ServiceNow for ticket management and Excel for tracking assets.” Q: What’s the difference between single-mode and multi-mode fiber? 👉 “Single-mode fiber is designed for long-distance communication using a laser, usually with yellow jackets. Multi-mode fiber is used for shorter distances, typically inside data centers, and uses LED light, often with orange or aqua jackets.” Q: How do you troubleshoot a server that won’t power on? 👉 “First, I would check the power connections and ensure redundant power supplies are connected. Then I would verify if the PDU is providing power. If hardware seems fine, I would check logs or iDRAC/ILO for errors. If needed, I’d escalate through the ticketing system.” Q: Are you comfortable working night shifts? 👉 “Yes, I understand this role requires evening and weekend shifts. I am flexible and used to working in a 24/7 environment.”

4.Rack & Stack 常用单词汇总 🔹 机柜 & 附件 Rack → 机柜 Rack Unit (U) → 机柜高度单位（1U=1.75 英寸） Rails → 导轨（安装服务器用的金属滑轨） Blanking Panel / Filler Panel → 挡板（填充空位，保证气流） Cable Management Arm (CMA) → 线缆管理臂（服务器后方，方便走线） Rack Mount Kit → 上架套件（螺丝、导轨、支架） 🔹 电源相关 PDU (Power Distribution Unit) → 电源分配单元 PSU (Power Supply Unit) → 电源模块 Power Cord → 电源线 Redundant Power Supply → 冗余电源 🔹 网络 & 线缆 Patch Panel → 配线架 Ethernet Cable (Cat5e/Cat6/Cat6a) → 网线 Fiber Optic Cable (Single-mode / Multi-mode) → 光纤线 SFP / SFP+ / QSFP → 光模块 Transceiver → 收发器（SFP 模块） Cable Label → 线缆标签 🔹 服务器安装相关 Rack & Stack → 服务器机柜上架 Unboxing → 开箱 Mounting Bracket → 安装支架 Thumbscrews / Cage Nuts → 手拧螺丝 / 笼式螺母 Grounding → 接地 🔹 散热 & 环境 Airflow → 气流 Hot Aisle / Cold Aisle → 热通道 / 冷通道 Cooling Fan → 风扇 Temperature Sensor → 温度传感器 📌 举例一句话（面试可能会问）： “When racking and stacking a server, how do you ensure proper airflow?” 👉 你就可以答： “I always install blanking panels in unused rack spaces and make sure cables are properly managed to avoid blocking airflow.”
5 
5.场景：主管让你把一台新服务器上架 Supervisor: “Hey, we need to rack and stack this new Dell server in Rack 12. Can you take care of it?” You: “Sure. I’ll start by unboxing the server and checking the rail kit.” Supervisor: “Good. Make sure you install the rails properly. Do you need any help?” You: “I think I’m good. I’ll mount the rails, slide the server into place, and secure it with cage nuts and screws.” Supervisor: “Don’t forget about power and network connections.” You: “Understood. I’ll connect both redundant PSUs to separate PDUs for power redundancy, and then connect the Ethernet and fiber cables. I’ll also label all the cables.” Supervisor: “Nice. How about airflow management?” You: “I’ll install blanking panels in empty rack spaces and make sure the cable management arm doesn’t block the airflow.” Supervisor: “Great. Once it’s powered on, log a ticket in ServiceNow to confirm installation.” You: “Will do. I’ll test the server power, check the iDRAC interface, and then update the ServiceNow ticket with the installation details.”

关键词汇回顾 Rail kit → 导轨套件 Cage nuts → 笼式螺母 Redundant PSUs → 冗余电源模块 PDUs → 电源分配单元 Blanking panels → 挡板 Cable management arm → 线缆管理臂 Label cables → 给线缆贴标签 ServiceNow ticket → 工单系统记录 
      
6.Rack & Stack Checklist (操作流程清单) 🔹 Step 1: Preparation – 准备工作 “Verify the rack location and check the rack space (U-size).” 确认机柜位置和高度（多少 U）。 “Unbox the server and check for rail kit, power cords, and accessories.” 开箱，检查导轨、电源线和配件。 “Ensure proper ESD protection before handling components.” 操作前做好静电防护。 🔹 Step 2: Install Rails – 安装导轨 “Mount the rail kit securely in the rack.” 把导轨牢固地装到机柜上。 “Make sure rails are level and aligned properly.” 确保导轨水平、对齐。 🔹 Step 3: Rack the Server – 上架服务器 “Slide the server into the rails carefully.” 小心地把服务器推入导轨。 “Secure the server with cage nuts and thumbscrews.” 用笼式螺母和手拧螺丝固定服务器。 🔹 Step 4: Connect Power – 接电源 “Connect redundant PSUs to separate PDUs for power redundancy.” 把冗余电源分别接到不同的 PDU。 “Verify that power cords are firmly connected.” 确认电源线连接牢固。 🔹 Step 5: Connect Network & Fiber – 接网络和光纤 “Connect Ethernet and fiber cables as per the diagram.” 按照拓扑图连接网线和光纤。 “Label both ends of the cables for easy identification.” 给线缆两端贴标签，方便识别。 “Route cables through the cable management arm.” 把线缆走线放入理线臂。 🔹 Step 6: Airflow & Blanking Panels – 散热与挡板 “Install blanking panels in unused rack spaces to maintain proper airflow.” 在空位安装挡板，保证气流。 “Check that cables do not block fans or airflow.” 确认线缆不会挡住风扇或气流。 🔹 Step 7: Power On & Test – 开机测试 “Power on the server and check LED indicators.” 开机，查看服务器指示灯。 “Access the iDRAC/iLO interface to confirm hardware health.” 通过 iDRAC/iLO 确认硬件状态。 🔹 Step 8: Documentation – 文档与工单 “Update the asset inventory in Excel.” 在 Excel 里更新资产信息。 “Close or update the ServiceNow ticket with installation details.” 在 ServiceNow 工单系统里更新安装信息并关闭工单。 📌 总结： 面试回答时，你可以说： 👉 “My rack and stack process includes preparation, rail installation, mounting the server, connecting redundant power and network cables, managing airflow with blanking panels, powering on, and finally updating documentation in ServiceNow.”
